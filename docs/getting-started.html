Getting started

Let's get you up and running as fast as possible.

Requirement

Gentleman requires a meta-model: a model that defines a language. The format used for the grammar is JSON.

Metamodel

The metamodel is where every Gentleman project begin. Given one, you will then be able to load or create any instance (model) of it and in the editor. A metamodel is defined as a Grammar. The Grammar is made of an aggregation of POS and additional properties. Those can be one of the following: Rule, Enum, Datatype.

Rule

Rules are the main components found in a Grammar. They represent a chunk of your grammar that can be reused, repeated or extended. As such, there are three types: Abstract, Concrete and Nested.
